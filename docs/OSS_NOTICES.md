## Release Compliance Gate

Do not publish an APK, MSI, container image, or binary release until:

1. Every redistributed binary in `docs/THIRD_PARTY_BINARY_AUDIT.md` has an
   exact upstream release tag or source commit.
2. The source URL for each copyleft/MPL component is listed and still
   reachable.
3. Required license texts and notices are packaged with the release artifact.
4. Any binary that cannot be traced to redistributable source is removed or
   replaced with a reproducible build path.
5. `scripts/secret-scan.sh .` and the relevant platform tests pass.
